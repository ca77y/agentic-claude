# Engineering Docs

The durable engineering knowledge for `ca77y-agentic`. This repository is an Obsidian vault; open the repository root in Obsidian to use the Task Board, templates, graph, and backlinks.

Toolkit usage — what the plugin is, how to install it, and what each agent does — stays in the root [`README.md`](../README.md). Repo-maintenance rules stay in the root [`CLAUDE.md`](../CLAUDE.md).

## Layout

- `PRODUCT.md` - what the toolkit is for, its boundaries, and where it is going
- `ARCHITECTURE.md` - how the plugin is put together and how the agents fit
- `issues/` - known problems with no identified solution on our side
- `specs/` - temporary specs for in-flight units of work
- `tasks/` - one Markdown story card per file (the board)
- `_templates/` - story, task-card, and spec scaffolds

See [`CLAUDE.md`](./CLAUDE.md) for placement and lifecycle rules.
